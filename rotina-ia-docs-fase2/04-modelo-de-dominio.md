# Modelo de Domínio

## Entidades centrais
- User
- Profile
- Goal
- Plan
- Routine
- Task
- ExecutionLog
- Review

## Regra de domínio importante
Plano não é tarefa. Plano é uma estrutura organizadora.

## Hierarquia ideal
- Goal
  - Plan
    - Routine
      - Task
        - Subtask
          - ExecutionLog
