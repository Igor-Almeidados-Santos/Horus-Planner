# Decisão Arquitetural

A API deve ser construída como API de domínio, não como API genérica.

## Diretriz
- não pensar só em tabelas
- pensar em comportamento do negócio
- pensar em ciclo de planejamento
- pensar em análise de execução
- pensar em replanejamento

O valor do sistema está nessa camada de inteligência operacional.
