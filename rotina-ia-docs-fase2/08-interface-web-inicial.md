# Interface Web Inicial

A interface do MVP deve ter 5 telas principais.

## 1. Dashboard
- visão geral do dia
- tarefas prioritárias
- tempo planejado
- tempo realizado
- progresso semanal

## 2. Objetivos
- listar objetivos
- ver progresso
- vincular planos

## 3. Plano
- estrutura do plano
- rotinas
- tarefas
- calendário lógico
- versão do plano

## 4. Execução diária
- tarefa atual
- iniciar/pausar/concluir
- registrar bloqueios
- registrar observações

## 5. Revisão
- resumo semanal
- gargalos
- tarefas atrasadas
- recomendações do agente
