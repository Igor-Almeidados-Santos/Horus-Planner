# Stack Recomendada

## Backend

- Node.js
- TypeScript
- NestJS ou Fastify

### Escolha prática

- **Fastify** para simplicidade e velocidade
- **NestJS** para modularidade, crescimento e organização corporativa

## Frontend

- Next.js
- TypeScript
- Tailwind CSS

## Banco de dados

- PostgreSQL no Cloud SQL

## ORM

- Prisma

## Autenticação

- JWT
- Refresh token
- evolução futura para OAuth Google

## Infra complementar

- Secret Manager
- Cloud Storage
- Cloud Run para deploy futuro
- Firestore como camada complementar opcional
