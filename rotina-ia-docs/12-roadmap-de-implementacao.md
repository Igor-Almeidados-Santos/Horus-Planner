# Roadmap de Implementação

## Fase 1 — MVP funcional

Construir:

- autenticação
- usuários
- objetivos
- planos
- tarefas
- execução
- dashboard básico
- integração com agente

## Fase 2 — inteligência operacional

Adicionar:

- replanejamento automático
- scoring de prioridade
- métricas de aderência
- revisão semanal automatizada

## Fase 3 — produto robusto

Adicionar:

- múltiplos perfis de rotina
- agenda visual avançada
- recomendações preditivas
- importação de calendário
- notificações
- análise longitudinal
