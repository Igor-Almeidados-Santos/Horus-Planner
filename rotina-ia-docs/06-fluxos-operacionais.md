# Fluxos Operacionais

## Fluxo 1 — criação de plano

1. O usuário informa um objetivo.
2. O agente interpreta o objetivo e coleta contexto.
3. O agente gera um plano estruturado.
4. A API valida e grava.
5. A interface exibe a estrutura final.

## Fluxo 2 — execução

1. O usuário inicia ou conclui tarefas.
2. A API atualiza progresso.
3. O sistema recalcula aderência e registra histórico.
4. O agente consome os dados atualizados.
5. O agente ajusta orientações futuras.

## Fluxo 3 — revisão

1. A API consolida métricas diárias e semanais.
2. O agente interpreta padrões.
3. A interface apresenta análise.
4. O sistema propõe replanejamento.
